# 使い方
1. annotationフォルダを作成し、その直下でgit cloneする。  
`mkdir annotation`  
`cd annotation`  
`git clone https://github.com/saityyy/make_animeface_dataset.git`  
2. 指定のディレクトリ構成になるようにフォルダやファイルを作成する。  
3. 取ってきたい画像が入ったフォルダのパスを設定する。
